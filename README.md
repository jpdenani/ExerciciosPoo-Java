📘 Exercícios de Programação Orientada a Objetos em Java

Este repositório contém exercícios desenvolvidos durante a disciplina de Programação Orientada a Objetos (POO) na faculdade, utilizando a linguagem Java.

O objetivo é praticar e consolidar os principais conceitos da orientação a objetos, aplicando-os em exemplos práticos e exercícios propostos em aula.

🎯 Objetivos

Praticar os fundamentos da Programação Orientada a Objetos

Aplicar conceitos como:

Classes e Objetos

Atributos e Métodos

Encapsulamento

Construtores

Organização de código em pacotes

Desenvolver lógica de programação utilizando Java

Criar soluções organizadas e reutilizáveis

A Programação Orientada a Objetos permite criar sistemas mais organizados, reutilizáveis e fáceis de manter, sendo um dos pilares do desenvolvimento moderno em Java. ()
